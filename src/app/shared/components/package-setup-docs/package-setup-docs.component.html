Setup

Install the package:

npm install @flxng/{{package}}

After importing the module the lib is ready to use:

{{importingModuleDemoCode}}


Versions Compatibility

The major lib version is compatible with major Angular version, for example:

Lib version Angular version
8.x.x 8.x.x
10.x.x 10.x.x


To install the specific major version of the lib (e.g. 10):

npm install @flxng/{{package}}@latest-10
